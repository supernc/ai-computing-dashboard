# 城市智算中心运营驾驶舱

> 国家新一代人工智能公共算力开放创新平台 · 数据可视化大屏

## 预览

在线访问：https://supernc.github.io/ai-computing-dashboard/

## 特性

- **16:9 标准大屏**：1920×1080 基准，自适应缩放
- **三行五区布局**：顶部 KPI + 四宫格数据面板
- **政务科技风格**：深蓝 #0A2342 + 科技青 #00E4FF
- **实时数据刷新**：每 4 秒自动更新模拟数据
- **纯静态部署**：HTML + CSS + ECharts，无需构建

## 技术栈

- HTML5 / CSS3 Grid
- ECharts 5.5
- Vanilla JavaScript
- GitHub Pages

## 本地运行

```bash
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080
```
